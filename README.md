# RPMLauncher (製作中)

## 介紹

這是一個使用 Flutter 框架與 Dart語言製成的 Minecraft啟動器，主要目的是要簡化安裝 Minecraft 的麻煩。

[巴哈姆特文章](https://forum.gamer.com.tw/C.php?bsn=18673&snA=193012&tnum=1)

## 特色功能
- 自動安裝並設定對應版本的Java
- 自動安裝模組載入器 (目前支援 Fabric，Forge仍在製作中)
- 在啟動器內可以直接從CurseForge、Modrinth等平台下載模組
- 支援監控遊戲日誌
- 支援多國語言
- 自動檢測資源檔案是否完整  
- 還有許多實用與方便的功能
